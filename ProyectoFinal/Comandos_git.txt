1. git branch -> para ver en que rama estamos y que ramas hay
2. git checkout "rama" -> para mover en una rama
3. git status -> para ver cambios pendientes de subir a git 
4. git add . -> para agregar los cambios al stage
5. git commit -m "mensaje del commit" -> para comprometer los cambios. 
6. git push origin "rama" -> subir cambios a github a la rama en la que estamos. 
7. git merge "rama desde la que quiero traer cambios" -> ojo debemos estar parados en la rama a donde queremos traer los cambios.
